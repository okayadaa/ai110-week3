The core concept is brainstorming and building structure for a food ordering app, collaborating with an AI agent. ## Most likely to struggle:
I believe the struggle that is most likely to be encounter is creating test cases specifically algorithm enhancement and implementing simplify logic for each four classes without involving any sort of complexity. The output must be accurate to verify each method behavior.

## AI assisting vs AI misleading:
With AI assistance, it operated properly with UML diagram and providing explanation of attributes and methods. Including relationships and design decisions. One thing that AI was misleading was when creating a small test case for compute total cost method. It provided an inaccurate calculation.

## One way guidance:
A couple of guidance that I would insist is either constructing or illustrating the structure to keep track of how the app should ideally operate. They should note that the spec file and design file have to align. Be very detail oriented when working with methods and classes but at the same time don't create complex logic.
